# Rediance — UIモック仕様書

SNS「Rediance」のUIプロトタイプです。プログラマがモックを見ながら仕様を理解するための資料です。

---

## プロジェクト概要

**コンセプト：「フォロワーの数だけが影響力じゃない」**

Radianceは、フォロワーをただカウントするのではなく、**誰にどう評価されているか**で影響力を測るSNSです。

フォロワーを5段階の階層（Circle）に分類し、上位Circleに入れてもらうほど影響力スコアが上がります。影響力の計算構造はGoogleのPageRankと同じ**再帰的な仕組み**です：

```
影響力スコア ＝ フォロワーの質（誰にフォローされているか）
             × Circle評価の深さ（どのCircleに入れられているか）
             ＋ 投稿のエンゲージメント
```

影響力の高い人にInnerに入れてもらうほど、自分のスコアも大きく上がります。

## 画面一覧と仕様

### 1. ホーム（タイムライン）— TikTok風フルスクリーン

- 投稿が1件ずつ全画面表示。上下スワイプで切り替え
- 各投稿はAIが自動生成した背景デザイン付き（モックではCSSグラデーション）
- 投稿テキストは画面中央に大きく表示
- 左下にユーザー情報＋称号を表示
- 下部にアクションバー：いいね・コメント・リポスト・インプレッション数

---

### 2. 投稿フロー — AIデザイン提案

1. テキストを入力
2. 「AIデザインを生成」ボタンを押す
3. AIが4パターンの背景を提案（サイバー／ナチュラル／エレガント／ポップ）
4. ユーザーが好みのデザインを選んで投稿
5. 投稿完了後、タイムラインの先頭に追加される

---

### 3. 「✨ 強い光が届きました」— Radiance投稿

フォローしていなくても、影響力が非常に高いユーザー（Sun）の投稿がタイムラインに流れてきます。

「強い光（影響力）は遠くまで届く」というコンセプトの体現です。ゴールドのグロー効果で視覚的に特別扱いします。

---

### 4. プロフィール — ポケモンカード風

プロフィールページは1枚のカードとしてデザインされています。称号（天体名）によってカードの外観が変わります：

| 称号 | ランク | カードデザイン |
|------|--------|--------------|
| ☀️ Sun（太陽） | 上位1% | ゴールド枠、キラキラアニメーション |
| 💫 Nova（新星） | 上位5% | 紫枠、紫グロー |
| 🌙 Moon（月） | 上位20% | シルバー枠、シルバーグロー |
| ⭐ Star（星） | 上位50% | ブルー枠 |
| 🕯️ Ember（残り火） | 下位50% | ダークブラウン |

カードの表示内容：
- 影響力スコア（数値）
- 「上位○%」表示
- 次の称号へのプログレスバー

**重要：** スコアはフォロワー数ではなく計算された影響力値です。1人フォロワーが減っても大きく変動しません。称号は全ユーザー相対評価なのでインフレしません。

---

### 5. Circle（フォロワー階層）— 5段階

| Circle | 上限 | 意味 |
|--------|------|------|
| 🔥 Inner | 最大3人 | 最も信頼する存在 |
| 💎 Trusted | 最大10人 | 信頼する仲間 |
| ⭐ Close | 最大30人 | 近い存在 |
| 🤝 Connected | 最大100人 | 繋がり |
| 👥 Ambient | 制限なし | 一般フォロワー（デフォルト） |

**影響力への重み付け：**
- 他人にInnerに入れてもらう → 最も高いポイント
- Ambientに入れてもらう → ポイントなし
- フォロワー自身の影響力も重みに反映（Sunの人にInnerに入れられる ≫ Emberの人にInnerに入れられる）

---

### 6. Circle設定画面

- 5階層を折りたたみ式で表示
- 「＋追加」ボタン → ボトムシートが開いてフォロワーを選択できる
- おすすめ表示＋検索バー付き
- 追加するとプロフィールカードに即反映

---

### 7. Circle組閣ウィザード（index.html）

初回セットアップ専用の画面です。

- ステップバイステップで上位Circle（Inner）から順にフォロワーを選んでいく
- 全員を分類する必要はない。上位層だけ選べば、残りは自動でAmbientになる

---

### 8. リール動画

- タイムラインにリール風の動画投稿も混在
- 暗い背景に▶️再生ボタンのサムネイル表示
- 実装時は動画再生機能が必要

## 技術メモ

| 項目 | 内容 |
|------|------|
| 現状のモック | HTML + CSS + JS、1ファイル完結、フレームワークなし |
| 実装想定 | React Native / Flutter 等のモバイルフレームワーク |
| 影響力スコア計算 | バックエンド側でPageRank的アルゴリズムを実装 |
| AIデザイン生成 | 画像生成AI（Stable Diffusion等）との連携を想定 |
| Circle管理・スコア更新 | リアルタイムではなくバッチ処理でOK（称号がリアルタイムで変動するとUXが悪い） |
